# 🚀 GenAI Social Media Intelligence Platform

An AI-powered analytics platform that transforms Reddit conversations into actionable business insights using Python, SQL, HuggingFace Transformers, and Tableau.

---

## 📌 Overview

Social media generates massive volumes of user-generated content every day. Extracting meaningful insights from these conversations is critical for understanding audience behavior, engagement drivers, and emerging trends.

This project analyzes Reddit discussions across multiple communities and applies Natural Language Processing (NLP) techniques to identify emotional patterns, measure engagement performance, and generate business intelligence reports.

The platform combines analytics, AI, data visualization, and automated reporting into a complete end-to-end intelligence solution.

---

## 🎯 Business Objective

To answer the following key questions:

* Which emotions drive the highest engagement?
* How does engagement vary across online communities?
* What content characteristics generate stronger audience interaction?
* How can social media conversations be converted into actionable business insights?

## 📊 Dataset

The dataset consists of Reddit discussions collected across multiple communities.

### Features

| Feature          | Description             |
| ---------------- | ----------------------- |
| post_id          | Unique post identifier  |
| subreddit        | Community name          |
| title            | Post title/content      |
| score            | Reddit score            |
| num_comments     | Total comments          |
| created_at       | Post creation date      |
| emotion scores   | Emotion probabilities   |
| dominant_emotion | Predicted emotion label |

### Emotions Analyzed

* Joy
* Fear
* Anger
* Sadness
* Surprise
* Disgust

## 📈 Key Insights

### 🔥 Fear Drives Engagement

Fear-related discussions generated the highest average engagement among all analyzed emotions, significantly outperforming positive emotional categories.

### 📢 Emotional Content Performs Better

Posts with stronger emotional signals consistently attracted more comments and higher interaction levels.

### 🌐 Community-Specific Patterns

Engagement behavior varied across Reddit communities, indicating that audience response is heavily influenced by community context.

### 📊 Stable Engagement Trends

Monthly engagement remained relatively stable while showing identifiable spikes during key discussion periods.
